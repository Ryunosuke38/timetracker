# Timetracker WebApp

Uses [rails](https://github.com/rails/rails) as webframework, [rom](http://rom-rb.org) as data crunching and persistence layer and [Angular2](https://angular.io) as frontend framework.

# Getting Started

```
  $ bundle install
  $ npm install
  $ rails s
```
